# referent

Минимальное приложение на Next.js

## Установка

```bash
npm install
```

## Запуск

Для запуска в режиме разработки:

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка

Для создания production сборки:

```bash
npm run build
npm start
```
